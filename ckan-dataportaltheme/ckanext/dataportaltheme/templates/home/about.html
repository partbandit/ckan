What is Data Portal Timisoara

Computer-readable information is given in a format that can be understood by a computer. This means that all relevant fields can be extracted and analyzed without a human contribution. Just because a data set is digitally accessible does not necessarily mean that it is readable for the computer (more information here). In the spirit of open data, data sets must be provided in unprotected file formats. Tabular data is provided as CSV or JSON format. Metadata should be excluded from the data files. These will be included in a zip file with the data file (see Data Pack below).

About the Data Portal application

The purpose of the Data Portal platform is to respond to three issues related to open data in Romania. First, the portal will open to the public the data used in the Code for Romania projects. They can be accessed and viewed by anyone interested. Secondly, it will define standards and data protocols for data sets published by Code4Romania that can be implemented by any interested institution to provide public data. Thirdly, the platform will provide a starting point for developing an open data community in Romania. Through Data Portal we will provide support (infrastructure, procedures, know-how) to volunteers interested in opening and cleaning public data in Romania.
